# FinTrack – Expense & Budget Tracker Mobile App

FinTrack is a cross-platform mobile application built using React Native that helps users track expenses, manage budgets, and visualize spending analytics.

## Features
- Login & Signup
- Add Expense
- Expense Dashboard
- Expense List
- Reports
- Offline Storage
- API Integration
- Error Handling
- Clean UI

## Tech Stack
- React Native
- React Navigation
- Context API
- REST APIs
- AsyncStorage
- Chart Kit

## Project Structure
- Screens
- Components
- Navigation
- Services
- Context State Management

## Setup Instructions
1. Clone the repository
2. Run npm install
3. Run npx react-native run-android
